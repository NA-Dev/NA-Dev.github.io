# My Data Science Portfolio Site
A multi-page React website

### Deploy frontend for development
```bash
cd frontend
unset HOST
nvm use 12
npm start
````

### Deploy to GitHub Pages
```bash
cd frontend
unset HOST
nvm use 12
npm run build
npm run deploy
```
